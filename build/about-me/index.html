My name is Sid Lacy. What is this website?

My intention behind swlacy.com is two-sided: first, I use this space to enrich my knowledge, providing a more elegant, professional location to publish work, which I can tailor to my specific needs. Second, I aim for this space to be of equal use to others, to become a trove of information those interested in computer security can use to benefit from as well.
